<h1 align="center">ExecutorX</h1>
<p align="center">
This is an online executor which is built around the <a href="https://github.com/engineer-man/piston" target="_blank">Piston API</a>. <a href="https://executor.edwinreji.com/" target="_blank">Live Demo</a></p>

<img src="./images/preview.png" width="100%"></img>

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/edw-in/ExecutorX
```

2. Install the dependancies

```bash
npm install
# or
yarn install
```

3. Run the development server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
